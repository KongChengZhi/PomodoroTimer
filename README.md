# 番茄钟 Chrome 插件

- 一个简单的番茄钟计时器 Chrome 插件，帮助你提高工作效率。
- Version 1.1

## 功能特点

- 25分钟工作时间
- 5分钟短休息
- 15分钟长休息
- 时间到会发送通知提醒
- 可暂停和重置计时器

## 安装说明

1. 打开 Chrome 浏览器
2. 进入扩展程序页面 (chrome://extensions/)
3. 开启开发者模式
4. 点击"加载已解压的扩展程序"
5. 选择本项目文件夹

## 使用方法

1. 点击 Chrome 工具栏中的番茄钟图标
2. 选择想要的计时模式（工作、短休息或长休息）
3. 点击"开始"按钮开始计时
4. 可以随时点击"暂停"或"重置"按钮
5. 时间结束时会收到系统通知

## 开发说明

本项目使用 cnpm 作为包管理器：

```bash
cnpm install
```
